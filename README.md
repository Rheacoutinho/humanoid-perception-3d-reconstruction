# Humanoid Internship Challenge — Video to 3D Reconstruction

A system that takes a short indoor video and reconstructs a geometrically coherent,
semantically labelled 3D scene.

## Pipeline
1. Frame extraction (OpenCV)
2. Pose estimation + dense reconstruction (MASt3R)
3. Depth refinement (Depth-Anything V2)
4. Mesh reconstruction (Open3D)
5. 2D segmentation (SAM2 + CLIP)
6. Label lifting to 3D
7. Interactive viewer (Gradio)
